# PG Admin

1. Create a database in pgAdmin
2. Run SQL script

# Backend

1. Create a folder backend
2. Run and install npm

```jsx
npm init -y
npm install express knex pg cors
```

1. Create file db.js
    
    Connection file with credentials to establish connection between backend and api
    
2. Start the server

```jsx
cd backend
node server.js
```

# Frontend

Start the front end

```jsx
cd frontend
npm run
```
